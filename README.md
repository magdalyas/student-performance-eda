# Student Performance EDA

## Overview

This project explores factors associated with student academic performance using Python, Pandas, and Seaborn. The dataset contains information on 1,000 students, including demographic variables, test preparation completion, and exam scores.

## Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib

## Analysis Performed

* Data inspection and quality checks
* Missing value and duplicate analysis
* Creation of an average score metric
* Distribution analysis of math scores
* Comparison of student performance by test preparation status
* Relationship analysis between reading and writing scores

## Key Findings

* Most students scored between 60 and 80 on the math exam.
* Students who completed a test preparation course generally achieved higher scores across all subjects.
* Reading and writing scores showed a strong positive relationship.
* The dataset contained no missing values or duplicate records.

## Dataset

Student Performance Dataset (1,000 observations)
